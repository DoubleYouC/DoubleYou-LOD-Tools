# DoubleYou LOD Tools
This Blender addon adds some useful tools that help in creation of LOD models.

## INSTALLATION

1. Download the add-on: [DoubleYou-LOD-Tools](https://github.com/DoubleYouC/DoubleYou-LOD-Tools/archive/master.zip).
2. Go to File > User Preferences > Add-ons.
3. Click on Install Add-on from File.
4. Choose `DoubleYou-LOD-Tools-master` archive.
5. Activate DoubleYou LOD Tools.
